### Basic usage

```js
const targetStyle = {
  color: "#f06",
  fontFamily: "-apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol'",
};

<Fade in={true}>
  <div style={targetStyle}>
    Fade me in
  </div>
</Fade>
```
